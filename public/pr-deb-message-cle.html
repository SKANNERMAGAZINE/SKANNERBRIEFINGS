SON MESSAGE

Une entreprise a mille choses à dire. Un journaliste n'en retient qu'une. Un message clé, c'est cette phrase que vous voulez voir ressortir — et qui doit survivre à la reformulation d'un tiers.

Niveau Débutant
Durée 13 min

01Qu'est-ce qu'un message clé

Un message clé n'est pas un slogan, ni votre argumentaire complet. C'est l'idée unique que vous voulez qu'un lecteur retienne — formulée de façon si claire qu'un journaliste peut la reprendre sans la déformer. La règle : un message, une idée.

→

Ce n'est pas un slogan. Un slogan séduit ; un message clé informe et se prouve. « L'énergie est notre avenir » est un slogan. « Nous produisons 40 % de notre électricité en renouvelable » est un message.

→

Ce n'est pas tout dire. Vouloir caser cinq idées, c'est n'en transmettre aucune. Le message clé assume de choisir.

→

Il doit survivre à la reformulation. Un tiers va le réécrire avec ses mots. S'il reste juste après reformulation, il est solide.

02Les cinq qualités d'un bon message

Un message clé se juge sur cinq critères. C'est exactement ce que le template vous fait noter — ils forment la grille de solidité.

ClartéCompréhensible d'emblée
Compris en une lecture, sans jargon. Si on doit le relire, il n'est pas clair.
PreuveÉtayé, pas affirmé
Un chiffre, un fait, un exemple concret le soutient. Sans preuve, c'est une opinion.
DifférenciationPropre à vous
N'importe quel concurrent pourrait-il dire la même chose ? Si oui, le message n'est pas à vous.
MémorabilitéÇa reste
Court, concret, une image ou un chiffre marquant. Ce qui ne se retient pas ne se reprend pas.
AudiencePour eux, pas pour vous
Formulé du point de vue du lecteur, pas de votre communication interne.
03Le test du tiers

La meilleure vérification est simple : faites répéter votre message à quelqu'un qui vient de l'entendre une fois. Ce qu'il/elle restitue, c'est ce qui passera dans la presse — pas ce que vous aviez prévu.

Votre message n'est pas ce que vous dites. C'est ce que l'autre retient et répète.

Test de solidité
À garder en tête. Un bon message porte une seule idée, s'appuie sur une preuve, vous est propre, se retient, et parle à l'audience. Ces cinq critères sont votre grille de contrôle.
À retenir
Une seule idée
Un message = une idée. Vouloir tout dire, c'est ne rien transmettre.
Prouvé, pas affirmé
Un chiffre, un fait, un exemple. Sans preuve, votre message reste une opinion.
Propre à vous
Si un concurrent pourrait dire la même chose, le message n'est pas différenciant.
Ce que l'autre retient
Votre message, c'est ce qu'un tiers restitue après une seule écoute — pas ce que vous aviez prévu.
Glossaire
Message cléIdée unique qu'on veut faire retenir, formulée pour être reprise sans déformation.
PreuveChiffre, fait ou exemple concret qui étaye un message et le distingue d'une simple affirmation.
DifférenciationCe qui rend un message propre à une marque : un concurrent ne pourrait pas le tenir tel quel.
MémorabilitéCapacité d'un message à être retenu : court, concret, porté par une image ou un chiffre marquant.
SloganFormule courte à visée de séduction ou de mémorisation. Ne se prouve pas, contrairement au message clé.
ArgumentaireEnsemble structuré des arguments d'une prise de parole. Le message clé en est la pointe, pas la totalité.
ReformulationRéécriture d'un message par un tiers (journaliste). Un bon message reste juste après reformulation.
Point de vue audienceFormulation qui part de l'intérêt du lecteur plutôt que de la communication interne de l'émetteur.
JargonVocabulaire d'initiés qui nuit à la clarté. À bannir d'un message destiné à une audience large.
Taux de rétentionPart d'un message effectivement retenue par l'audience. Chute quand le message porte plusieurs idées.
AccrocheFormulation d'entrée qui capte l'attention et donne envie de lire la suite.
Message principalLe message central d'une prise de parole, que viennent soutenir des messages de preuve secondaires.
